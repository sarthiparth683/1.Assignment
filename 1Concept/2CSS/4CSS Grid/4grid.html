Grid Line Naming (just for Naming convention.)

In CSS Grid, you can name the lines of the grid using the 'grid-template-columns' and 'grid-template-rows' properties, and then reference these names in the 'grid-column' and 'grid-row' properties.

This allows you to create more meaningful and reusable grid layouts and makes it easier to understand and maintain your code.

Example
.container {
display: grid;
grid-template-columns: [start] 1fr [mid] 2fr [end];
grid-template-rows: [top] 100px [bottom];
}

.item {
grid-column: start / end;
grid-row: top / bottom;
}
