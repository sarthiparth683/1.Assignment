Events handlers Event handlers allow you to react when certain things (an event) happen to the page on an element on the page. There are many different types of events that can occur. For example: The user selects a certain element or hovers the cursor over a certain element. The user chooses a key on the keyboard. The user resizes or closes the browser window. A web page finishes loading. A form is submitted. A video is played, paused, or finished. An error occurs.
