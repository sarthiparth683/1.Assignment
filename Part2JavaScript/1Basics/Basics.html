    Comparison operators
  1. = (single equal to only Assigns Value, it doesnot compare)
  2. == (double equal to) compares only value not data Type
  3. != (Not equal to gives opposit of (==) value and compare value only)
  4. === (triple equal to) compares both dataType and value
  5. !== (not double equal to) gives opposit value of (===).
