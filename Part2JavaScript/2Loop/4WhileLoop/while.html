While Loop

Same as for loop only syntax change's.


// While Loop concept

let sum = 0;
let i = 1; // initialiazation condition or starting
let n = 5;

while (i <= n) { // condition to stop or Ending condition
sum = sum + i; // logic or code
i++; // inc/dec
}
console.log(sum);

Loop control statement :- break and continue

break;(use to breal or stop the loop)

continue;(use to continue the loop but skipping itself)
