For Loop

1.) for (let i=1; i<=9; i++) {
console.log("Assending Order"); }

2.) for(let i=9; i>=0; i--) {
console.log("Descending Order or Reverse"); }

console inside loop(otuput repeats), and console outside loop(only one output)
for counting it is count = count + 1 not i;
for adding string it is sum = sum + num;

Loop helps us to avoid repeatation

    For Loop Follows 4 step
  1. Initialization or starting point
  2. Condition to stop(where loop will stop or end)
  3. Increment or decrement
  4. Code or logic

for (starting point; condition to stop the loop; inc/dec) {code/logic}

Loop control statement :- break and continue

break;(use to break or doesnot stop the loop)

continue;(use to continue the loop but skipping itself)
