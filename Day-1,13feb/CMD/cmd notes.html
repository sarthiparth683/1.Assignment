command line interface

mkdir/md - creates new folder - mkdir NewFolder cd - changes current directory - cd folderName cd.. - to go back from the directory dir - display all files and folder's name - dir type - open the file - type MyFile cat - use to display content of that file man - use to display manual page or documentation rmdir/rd - removes directory - rmdir NewFolder del/erase - deletes a file - del MyFile cls - to clear screen ipconfig - display network ls - to list the contents files and folder - ls touch - to create new file - touch file.txt nano - use to edit files cp - use to copy - cp file.txt rm - use to remove or delete file -rm file.txt mv - use to move or rename a file or directory - mv file.txt grep - use to search specific pattern or keyword - grep -r 'keyword' directory/ chmod - use to change the permission of a file or directory - chmod +x script.sh history - use to display previously executed commands
